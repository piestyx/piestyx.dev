About the Project

Project ECHO started as an idea for a home-assistant that then turned into an emergent character development and narrative-driven AI system bringing characters like Orielle to life. So far she is able to read her environment, react to player events and speeak out her reasonsed responses in real time. In the back-end it's been kept modular with the runtime orchestrating the individual pipelines.


She is also able to read and interact with chat from each of the 3 main platforms (Twitch, Youtube and Kick) but is not yet able to respond as this forms part of future development phases.
