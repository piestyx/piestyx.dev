About the Project

Project ECHO is a narrative-driven AI system that brings characters like Orielle to life. She reads her environment, reacts to player events, speaks in real time, and evolves through chat. Every component is built with privacy, control, and customization in mind.

Core Repositories

  • Lain Runtime — Orchestrates perception, inference, and avatar output
  • QudLog Exporter — Game mod for event export
  • Chatbot Pod — Minimal local LLM service container
